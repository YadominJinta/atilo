# Atilo

在Termux安装Linux



## 安装

``` bash
apt update && apt install git python python-pip -y
git clone https://github.com/WTNLXTBL/atilo
cd atilo
chmod +x atilo init_atlio
./init_atilo
./atilo help
```

## 使用方法

``` bash
Atilo           2.2.0
Usage: atilo [命令] [参数]

Atilo 是一个用来帮助你在termux上安装不同的GNU/Linux发行版的程序

命令:
images           列出可用镜像
remove           移除本地的镜像
pull             拉取远的镜像
run              运行镜像
clean            清除缓存
help             帮助
```

## 相关项目
